Reset Homebridge Accessory
  • This action is irreversible. Please read carefully before proceeding.
  • A reset will unpair this Homebridge instance from your Apple HomeKit setup.
  • You will need to remove the existing Homebridge accessory from the Home app manually.
  • All automations and will need to be reconfigured after a reset.
  • Your Homebridge username and pin will be changed.
  • The rest of your config will not be changed. If Homebridge is not starting due to a bad config a reset will not fix it.
